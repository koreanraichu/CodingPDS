<h2>Restriction enzyme cutter</h2>
대충 NEB cutter같은 겁니다. DNA 시퀀스를 입력하면 해당 시퀀스를 자르는 제한효소를 전부 보여줍니다. <br>
<ol>
<li>입력한 시퀀스를 자르는 제한효소를 1컷, 2컷, 3컷 이상(멀티컷)으로 보여줍니다. </li>
</ol>
